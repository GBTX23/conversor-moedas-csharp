#  Conversor de Moedas em C#

Projeto desenvolvido em C# com o objetivo de praticar lógica de programação e estruturas condicionais.

# Funcionalidades

* Conversão entre Real, Dólar, Euro, Libra, Iene e Franco suíço
* Menu interativo com múltiplas opções de conversão

# Tecnologias

* C#
* .NET

# Objetivo

Aprimorar lógica de programação e evolução na área de desenvolvimento.

# Status

Projeto em desenvolvimento — melhorias futuras serão adicionadas.
